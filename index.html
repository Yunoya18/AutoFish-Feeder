บทคัดย่อ

 โครงงานนี้เป็นการใช้ Microcontroller ในการสร้างโปรแกรมควบคุมระบบต่าง ๆ ที่ใช้ในการดูแลตู้ปลา เนื่องจากปลาต้องการการดูแลอย่างสม่ำเสมอ จึงต้องการทำให้สะดวกต่อการดูแลของผู้เลี้ยงและป้องกันเวลาที่ไม่ว่างให้อาหารปลาหรือลืมให้อาหารปลา ประกอบด้วย ระบบให้อาหารอัตโนมัติ ระบบตรวจสอบปริมาณอาหาร ระบบวัดอุณหภูมิของน้ำพร้อมแจ้งสถานะ โดยทั้งสามระบบนี้สามารถตรวจสอบข้อมูลและตั้งค่าได้ทาง website ซึ่งอาศัยการอ่านค่าที่ได้รับจาก Ultrasonic Sensor และ Temperature Sensor ร่วมกับ Servo

อุปกรณ์

water temperature sensor

micro servo motor sg90

hc-sr04 ultrasonic sensor

arduino uno r4 wifi

Function

  • ระบบให้อาหารอัตโนมัติ
  • ระบบตรวจสอบปริมาณอาหาร
  • ระบบวัดอุณหภูมิน้ำพร้อมแจ้งสถานะ

Input

  • ค่าอุณหภูมิจาก Waterproof Temperature Sensor
  • ค่าระยะห่างที่วัดได้จาก Ultrasonic Sensor

Output

  • ค่าที่ส่งไปยัง Servo
